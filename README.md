# PHP WordPress Developer Portfolio

A responsive one-page portfolio template for a PHP WordPress Developer, ready to host on GitHub Pages.

## What is included

- `index.html` — main portfolio page
- `styles.css` — responsive styling
- `main.js` — mobile menu, current year, scroll reveal animation
- `.nojekyll` — helps GitHub Pages serve static assets as-is

## How to customize

Open `index.html` and replace these placeholders:

- `Your Name`
- `YourName`
- `your.email@example.com`
- `github.com/yourusername`
- `linkedin.com/in/yourusername`
- Project titles, descriptions, links, testimonials, and statistics

You can also edit colors in `assets/css/styles.css` under the `:root` variables.

## How to deploy on GitHub Pages

1. Create a new GitHub repository, for example: `portfolio`.
2. Upload all files from this folder to the repository.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save. Your site will be available at:

```text
https://yourusername.github.io/portfolio/
```

For a user site, name the repository:

```text
yourusername.github.io
```

Then the site will be available at:

```text
https://yourusername.github.io/
```
